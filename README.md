### 필수 사항

- node >=12.14.0

---

### localhost 실행 방법

1. package를 설치합니다.

```shell script
yarn install
```

<br />

2. `yarn start`로 [http://localhost:3000](http://localhost:3000)로 접속합니다.

```shell script
yarn start
```

<br />

3. 테스트는 기본적인 url routing과 CustomTextArea.tsx에 대해서만 진행했습니다. `yarn test`로 테스트 결과를 확인할 수 있습니다.

```shell script
yarn test
```
